Configure speech-to-text (STT) and text-to-speech (TTS) used for voice messages.

{# ── TTS ── #}
{# Edge TTS voice picker #}

Microsoft Edge neural voices. Examples: en-US-AvaNeural, en-US-JennyNeural, en-GB-SoniaNeural.

{# Kokoro voice picker #}

Voice prefix sets language: af/am = English, ff = French, jf/jm = Japanese, if/im = Italian, etc.

Voice name sent to the remote API. Kokoro sidecar uses Kokoro names; other APIs use their own.
